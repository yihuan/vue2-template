# Vue 2.7 template

Vue 2.7 + Vue Router 3 + Pinia + ESLint + Prettier template project

## 项目启动

### 本地开发

- 包管理工具：yarn
- 本地安装：`yarn` or `yarn install`
- 本地删除重新安装：`yarn reinstall`
- 本地启动：`yarn start`
- 编译打包：`yarn build`
- Lint：`yarn lint`
